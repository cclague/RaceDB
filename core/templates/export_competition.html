Export a Competition (and all its Dependencies)

Exports the Competition and all its dependent data including:

  • Race Discipline and Class
  • Mass Start and TT Events with Start Waves
  • Category Format and Categories
  • Category Numbers
  • Report Labels
  • Participants and Participant Options (for Optional Events)
  • License Holders
  • Season's Pass Holders
  • Bib Numbers and Number Set
  • Signed Waivers

If "Export as Template" is selected, participant information will be excluded. This makes it easy to export existing competitions to use them as a template for future competitions.

If "Remove FTP Upload Info" option is selected, FTP host and login information will be excluded.
